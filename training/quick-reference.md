## Common Actions

### Search Records
1. Go to any dashboard
2. Use the search field
3. Enter keywords
4. Press Enter

### Filter Data
1. Click Filter icon
2. Select criteria
3. Click Apply
4. Click Clear to reset

### Export Data
1. Apply filters (optional)
2. Click Export
3. Choose format
4. Download file

### Manual Sync
1. Go to KWikSync Hub
2. Click Sync Now
3. Monitor in Sync Log
