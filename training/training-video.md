## Quick Start Guide

While the video is in production, here's a quick overview:

### 1. Access KWikSync
- Open ServiceNow Application Navigator
- Type "KWikSync" in the filter
- Select any module to begin

### 2. Configure Connection
- Navigate to **Connection Settings**
- Enter your Customer Key
- Test the connection

### 3. Explore Dashboards
- Start with **KWikSync Hub** for overview
- Navigate to specific dashboards as needed
- Use filters to find specific records

### 4. Get Help
- Visit the **Support** module
- Contact ServiceNowSupport@kw-corp.com
